# Proyecto: Soluciones estilo LeetCode

## Objetivo

Este repositorio contiene soluciones de problemas de algoritmia estilo "LeetCode" orientadas a nivel "Medium", adecuadas para entrevistas técnicas. Cada ejercicio incluye una resolución inicial (análisis y enfoque práctico) y una solución ideal optimizada.

Los problemas se centran principalmente en manipulación de arrays y strings, aunque también cubren otros temas comunes de entrevistas (listas enlazadas, árboles, búsquedas, programación dinámica, etc.).

## Estructura del repositorio

- `exercises/` — Carpeta principal con los ejercicios.
  - Cada ejercicio vive en su propia subcarpeta, por ejemplo `01_groupAnagrams/`.
  - Dentro de cada subcarpeta se incluyen implementaciones, variantes optimizadas y pruebas mínimas.

## Formato de las soluciones

- Presentación en dos fases:
  1. Resolución inicial: explicación del razonamiento, enfoque claro y código funcional.
  2. Solución ideal: mejoras en complejidad temporal/espacial, limpieza del código y casos borde.
- Comentarios concisos que expliquen las decisiones algorítmicas.

## Cómo contribuir

- Añadir una nueva carpeta con el número del ejercicio y un nombre descriptivo.
- Incluir al menos dos archivos: una primera solución y una versión optimizada.
- Añadir ejemplos de entrada/salida y, cuando sea relevante, análisis de complejidad.

## Uso rápido

1. Abrir la carpeta del ejercicio que te interese.
2. Revisar la resolución inicial y la solución optimizada.
3. Ejecutar el archivo con Node.js si corresponde (por ejemplo `node agroupAnagrams.js`).

## Notas

- El objetivo es practicar y preparar entrevistas técnicas con problemas representativos de nivel medio.
